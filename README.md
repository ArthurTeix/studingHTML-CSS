# 📘 Front-End Jornada de Aprendizado

Este repositório foi criado com o objetivo de documentar, de forma estruturada, minha evolução nos estudos de **desenvolvimento Front-End**, reunindo anotações, exercícios práticos e projetos desenvolvidos ao longo do processo.

---

## 📍 Sobre o projeto

A proposta deste repositório é servir como um **registro contínuo de aprendizado**, permitindo acompanhar minha progressão técnica, organização de código e consolidação de boas práticas no desenvolvimento web.

---

## 🧠 Conteúdos abordados

### 🔹 HTML

* Estruturação de páginas web
* Uso de tags semânticas
* Organização e hierarquia de conteúdo

### 🔹 CSS

* Conceitos fundamentais de estilização
* Seletores e propriedades
* Pseudo-classes e interação com o usuário
* Boas práticas de organização de estilos

---

## 🎯 Objetivos

Este projeto faz parte da minha jornada para me tornar um desenvolvedor **Full Stack**, com foco inicial na construção de uma base sólida em Front-End.

### 📌 Próximos passos:

* Aprimoramento em JavaScript (ES6+)
* Tipagem estática com TypeScript
* Desenvolvimento de interfaces com React
* Integração com APIs
* Introdução a banco de dados (MongoDB)
